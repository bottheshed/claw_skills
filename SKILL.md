# Domain Finder

Find and validate `.ai` domain names for your projects. This skill operates in three modes designed for rapid brand discovery.

## Response Format

When you ask for domains, I will show you **one brief example per mode**, then ask which you want to use. Max one paragraph. Example: **DIRECT:** "Check openfield.ai, openregistry.ai, opendeck.ai" → Results: ✅ openfield.ai (available), ❌ openregistry.ai (taken). **SMART:** "Give me words like 'flow sync peak'" → I generate 20+ combos (flowsync.ai, clearsync.ai, peaksync.ai) + check them. **AUTO:** I auto-generate 50+ startup names and return only available ones. Pick a mode and I'll run it.

## Mode Details

**DIRECT:** You provide domain names; I check them via WHOIS/registrar. Fast verification of names you already like.

**SMART:** You provide 2-3 concept words; I generate combinations (word pairs + pairs with relational words like "open", "clear", "peak", etc.). Returns only brandable, pronounceable names meeting noun requirements.

**AUTO:** I auto-generate 50+ startup-style names and check availability. Good for finding fresh options when you don't have a concept yet.

All results show only domains **available for purchase** via registrar (confirmed real-time). `.ai` TLD unless otherwise specified.
