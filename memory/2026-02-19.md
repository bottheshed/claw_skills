### Key Design Decisions
1. **No arbitrary mashups**: "sunbee" → out. "bluebird" → in.
2. **Color + noun first**: 0.45 weight in scoring
3. **Real prefixes only**: "out" + "field" = "outfield" (real word)
4. **Short noun pairings only**: "box" + "star" = "boxstar" (8 chars, feels right)
5. **Single word anchors**: 5-8 char branded nouns (e.g., "stripe", "slack")
